# Minik Hayatlar: Evim

Tarayıcıda çalışan özgün yaşam simülasyonu prototipi.

## Çalıştırma

1. Klasörü indirin veya zipten çıkarın.
2. `index.html` dosyasını modern bir tarayıcıda açın.
3. Yeni oyun başlatıp karakter yaratın.

## Özellikler

- HTML/CSS/JavaScript ile tek sayfa web oyunu
- Üstten 2D görünüm, Canvas çizimi
- Karakter yaratma
- Yaşam / İnşa / Eşya modu
- Izgara tabanlı zemin, duvar, kapı ve eşya sistemi
- Yatak, buzdolabı, tuvalet, duş, TV, bilgisayar, kitaplık, dekor ve spor eşyaları
- Nesneye tıklayınca etkileşim menüsü
- A* pathfinding ve komut sırası
- Otonom ihtiyaç davranışları
- Açlık, enerji, hijyen, tuvalet, eğlence, sosyallik, konfor, ortam ihtiyaçları
- Ruh hali, moodlet, beceri, kariyer, ilişki ve para sistemleri
- Gün/gece döngüsü, hız kontrolleri, iş saatleri, faturalar
- LocalStorage kayıt/yükleme ve otomatik kayıt
- Mobil uyumlu arayüz
